사냥터출처

10~17 : 하늘과 가까운 곳
17~20 : 거대한 나무
20~30 : 초록나무줄기
30~65 : 테마던전 엘리넬, 골드비치, 리에나해협(테라버닝은 엘리넬 빼고 2개만)
65~71 : 조용한 습지(자리 없으면 동굴 낭떠러지)
71~77 : 하늘계단1(자리 없거나 X축 스킬 직업이면 검은 빛의 정원2)
77~81 : 모래성 놀이터
81~89 : 갱도4(윗층 같이 못때리면 얼음골짜기2)
89~95 : 사헬지대2, 잠자는 사막(자리 없으면 연구소103호)
95~100 : 관계자 외 출입금지, 사헬지대2(Y축 좋은 직업이면 연구소 C-3)

100~107 : 리프레 동쪽 경계
107~109 : 몬스터파크 2판
109~110 : 노말자쿰 팔까지 다 잡기
110~115 : 장난감공장<6공정>3구역
115~125 : 잊혀진 회랑(자리 없으면 뒤틀린 회랑, 날카로운 절벽3or4)
125~ : 아랫마을 현상수배 월묘부터 여우까지만
~ 145 : 시련의 동굴1, 2, 3 잡히는 거 보고 순차적으로 이동하면서 사냥
145~150 : 커닝타워 2층 카페<3>
150~153 : 암벽거인 퀘스트 중간까지만 깨기(벌떼의 본거지 사냥터 때문에)
153~180 : 커닝타워 2층 카페<4>(자리 없으면 버닝 5단계 이상 벌떼의 본거지)
- 2층 카페<4>에서 사냥할거면 175까지만 찍어주고 복도203으로 이동
180~190 : 복도203, 기사의 전당2(윗점 있으면 기사의 전당2가 월등히 좋음)
190~200 : 적막한 바위길(자리 없으면 칼바람의 묘지. 최근 패치로 포탈 2개 더 생김)
